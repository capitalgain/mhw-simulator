# MHW:ICEBORNE スキルシミュ

「モンスターハンターワールド：アイスボーン」のスキルシミュレーター


## 概要

線形計画法を使用したスキルシミュレーターです。

アルゴリズムは下記を参考にし、実装しています。  
http://nap.s3.xrea.com/lpsim-v3.pdf

## 開発

コードは GPLv3 ライセンスの元、公開しています。  
イシューやプルリクをお待ちしています。

### 構成

- Node.js / npm
- TypeScript
- React / Redux

### 実行

```bash
npm install
npm run dev
```
