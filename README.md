# MHW:ICEBORNE スキルシミュ

「モンスターハンターワールド：アイスボーン」のスキルシミュレーター


## 概要

線形計画法を使用したスキルシミュレーターです。  

アルゴリズムは下記を参考にし、実装しています。  
http://nap.s3.xrea.com/lpsim-v2.pdf

5chのスキルシミュレーター開発スレの皆様に感謝


## 使い方

スキルを選択し、検索を押してください。
